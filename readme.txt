== Installation ==

== Installation ==

**1. Generate your API key on Packlink PRO**

Go to the Settings section at the top right corner and click on \"WooCommerce Module\" located in the left menu, and generate your key.

 

**2. Download and set up the module on your Wordpress account**

- Click on the url <a href="https://wordpress.org/plugins/packlinkpro/" target="_blank" title="Wordpress Packlink">https://wordpress.org/plugins/packlinkpro/</a> and  download the free Packlink PRO plugin in a zip format. 

- From your back-office, upload Packlink PRO plugin zip file. (Click on the « Add New » button then « Upload Plugin »)

- Click on « Packlink » tab in the WooCommerce section > Settings of your Wordpress, copy/paste your API key from Packlink PRO and click on « save changes ».

 

**3. Use of the module**

From now, when a new order has been successfully paid or you have accepted the payment, the order data will automatically be imported into your Packlink PRO account,

so that Packlink PRO is always updated with all orders that are ready for shipment in WooCommerce. To ship these orders, go to Packlink PRO, where you will be able to see

all your orders in the list of shipments. For each order you will need to select the shipping service.

 

Click <a href="https://support-pro.packlink.com/hc/es-es/articles/210158585" target="_blank" title="support">here</a> to get more information about the installation of the module.
